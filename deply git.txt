

criar uma pasta chamada repositorio do lado da pasta do projeto dentro do servidor

mkdir repositorio

entrar na pasta repositorio

cd repositorio

usar comando git init dentro da pasta repositorio

git init --bare

quanto o pull der problema executar o seguinte 

git stash
git stash drop
git pull


alterando arquivo do nginx

nano /etc/nginx/nginx.conf


mudar a linha abaixo para aumentar o tamnho da imagem que pode ser enviada para o servidor

client_max_body_size 100M;
